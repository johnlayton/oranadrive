## sumo

### Setup oh-my-zsh

```zsh
pushd $ZSH/custom/plugins && \
  git clone git@github.com:johnlayton/oranadrive.git psqlx && \
  popd || echo "I'm broken"
```

```zsh
plugins=(... sumo)
```

### Setup other

```zsh
pushd $HOME && \
  git clone git@github.com:johnlayton/oranadrive.git .psqlx && \
  popd || echo "I'm broken"
```

```zsh
source ~/.psqlx/psqlx.plugin.zsh
```
